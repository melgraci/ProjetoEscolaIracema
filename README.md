# Site da Escola Iracema Rodrigues

## Descrição
Site institucional da Escola Iracema Rodrigues, desenvolvido para apresentar a história da escola, seus projetos, eventos e sua história. 

## Tecnologias utilizadas
- HTML / CSS
- JavaScript
- VS Code

## Funcionalidades
- Exibição de informações institucionais
- Destaques da escola (excelência acadêmica, comunidade escolar, projetos)
- Últimas notícias e eventos
- Estrutura responsiva para diferentes dispositivos
